Продвинутые Аудио Темы и Фонетическая Пунктуация (Audio Themes NG & Phonetic Punctuation)

Это дополнение обеспечивает захватывающий звуковой опыт для пользователей программы экранного доступа NVDA, воспроизводя звуки для различных событий пользовательского интерфейса. Оно позволяет создавать, устанавливать и настраивать звуковые темы, улучшая звуковой отклик от системы и приложений.

Основные возможности

  • Звуковые эффекты: Воспроизводит звуки для событий пользовательского интерфейса, таких как фокусировка на элементах управления, навигация по спискам, открытие/закрытие окон и многое другое.
  • 3D Аудио: Использует Steam Audio для обеспечения трехмерного позиционирования звука, давая точное представление о том, где именно на экране находятся элементы управления.
  • Реверберация: Добавляет эффекты реверберации и акустики помещения для создания более реалистичной и глубокой атмосферы.
  • Настраиваемые темы: Позволяет пользователям легко создавать, устанавливать и переключаться между различными звуковыми темами.
  • Студия Аудио Тем (Audio Themes Studio V2): Встроенный инструмент для создания новых или редактирования существующих звуковых тем с возможностью прямой записи с микрофона или с помощью перетаскивания (Drag & Drop).
  • Продвинутые звуки набора текста (Advanced Typing Sounds): Имитирует звуки реальной физической клавиатуры с 3D-позиционированием, динамическим изменением громкости в зависимости от скорости печати и специальными звуками для клавиш Enter, Backspace, Space, Shift, Ctrl, Alt.
  • Звуковой маяк / Сонар (Audio Beacon / Sonar): Позволяет установить звуковой маяк в любом месте экрана и перемещаться, слыша в реальном времени сигналы сонара, ориентирующие вас относительно маяка.
  • Продвинутая навигация: Интегрированные движки SentenceNav и BrowserNav для удобной навигации по тексту и веб-страницам без конфликта со стандартными клавишами со стрелками.

Установка

  1. Скачайте последний релиз дополнения с официального канала разработчика.
  2. Откройте скачанный файл с расширением .nvda-addon.
  3. NVDA попросит подтвердить установку. Выберите “Да” (Yes).
  4. Перезапустите NVDA для завершения установки.

Как использовать

Включение и выключение звуковых тем

Вы можете включить или отключить функцию звуковых тем в настройках NVDA:

  1. Откройте меню NVDA (NVDA+N).
  2. Перейдите в “Параметры” (Preferences) -> “Настройки” (Settings).
  3. В диалоговом окне настроек выберите категорию “Продвинутые Аудио Темы” (Advanced Audio Themes).
  4. Установите или снимите флажок “Включить звуковые темы” (Enable audio themes).

Выбор и управление темами

В панели настроек “Аудио Темы” вы можете:

  • Выбрать тему: Выберите из списка установленных звуковых тем.
  • Добавить новую тему: Нажмите кнопку “Добавить новую...” (Add New...), чтобы установить пакет темы из файла .atp.
  • Удалить тему: Выберите тему и нажмите кнопку “Удалить” (Remove).
  • О теме: Нажмите кнопку “О теме” (About), чтобы просмотреть информацию о выбранной теме.

Использование Студии Аудио Тем V2

Студия Аудио Тем позволяет создавать и редактировать звуковые темы. Чтобы открыть студию:

  1. Откройте меню NVDA (NVDA+N).
  2. Выберите “Студия Аудио Тем” (Audio Themes Studio).

В студии вы можете:

  • Создать новую аудио тему: Пошаговое руководство по созданию новой темы с нуля.
  • Настроить существующую тему: Изменить звуки уже установленной темы.
  • Запись с микрофона: Теперь вы можете записывать свой голос или любой звук прямо с микрофона, чтобы назначить его на событие пользовательского интерфейса!
  • Перетаскивание (Drag & Drop): Вы можете перетаскивать аудиофайлы прямо в окно Студии для их быстрого назначения.

Экспорт вашей темы

После создания или редактирования темы вы можете экспортировать ее в виде файла .atp, чтобы поделиться ею с другими. Опция экспорта находится на экране редактирования в Студии.

Продвинутые правила и фонетическая пунктуация

Звуковые значки и правила речи позволяют NVDA воспроизводить звуковые сигналы (earcons), а/также использовать другие речевые эффекты, такие как изменение просодии.

Использование

  1. Убедитесь, что функция включена. Нажмите NVDA+Alt+P для переключения.
  2. Правила можно настроить в специальном диалоговом окне в меню параметров NVDA.
  3. По умолчанию вам доступен набор предустановленных звуковых правил.
  4. Правила сохраняются в файле earconsAndSpeechRules.json в каталоге пользовательской конфигурации NVDA.

Комбинации клавиш

  • NVDA+Alt+N: Включение/выключение звуковых тем. (Быстрое двойное нажатие включает/выключает звуки набора текста).
  • NVDA+Alt+T: Циклическое переключение между доступными звуковыми темами.
  • NVDA+Alt+Y: Циклическое переключение между доступными пакетами звуков набора текста.
  • NVDA+Alt+K: Включение/выключение звуков набора текста.
  • NVDA+Shift+B: Установка/удаление звукового маяка (Beacon) на текущем объекте навигатора для ориентации с помощью сонара.
  • NVDA+Shift+A: Вход в командный слой звуковых тем (нажмите это сочетание, затем клавишу команды: 'h' для справки, 't' для переключения тем, 'p' для правил, 's' для уровня подробности, 'c' для уровня заголовка, 'o' для порядка речи).
  • NVDA+Alt+P: Включение/выключение звуковых значков и правил речи.
  • NVDA+Alt+[ (левая квадратная скобка): Переключение режима краткого/подробного сообщения о состоянии.
  • NVDA+H: Озвучить текущий уровень заголовка.
  • NVDA+Tab: Сообщить об объекте под курсором с полными координатами 3D-аудио.
  • Alt+Стрелки: Продвинутая навигация по предложениям и фразам (SentenceNav).
  • NVDA+Alt+Стрелки: Продвинутая веб-навигация (BrowserNav).
