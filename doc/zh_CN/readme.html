高级音效主题与语音规则 (Audio Themes NG & Phonetic Punctuation)

本插件通过为各种用户界面 (UI) 事件播放音效，为 NVDA 屏幕阅读器用户提供沉浸式的音频体验。它支持创建、安装和自定义音效主题，从而增强来自操作系统和应用程序的听觉反馈。

主要功能

  • 界面音效： 为聚焦控件、浏览列表、打开/关闭窗口等 UI 事件播放音效。
  • 3D 空间音效： 采用 Steam Audio 提供 3D 定位音效，让您清晰感知控件在屏幕上的具体位置。
  • 混响与环境模拟： 为音效添加混响与房间声学效果，带来更具深度和真实感的沉浸体验。
  • 自定义主题： 允许用户轻松创建、安装并在不同的音效主题之间进行切换。
  • 音效主题工作室 (Audio Themes Studio V2)： 内置的可视化工具，支持通过麦克风直接录音或拖放 (Drag & Drop) 音频文件来创建新主题或编辑现有主题。
  • 高级打字音效 (Advanced Typing Sounds)： 模拟真实物理键盘的按键音，具备 3D 空间定位、随打字速度动态调节音量以及针对特殊按键 (Enter、Backspace、Space、Shift、Ctrl、Alt) 的专属音效。
  • 音频信标 / 声纳 (Audio Beacon / Sonar)： 允许在屏幕任意位置放置音频信标，在移动浏览时发出实时声纳脉冲，引导您了解当前位置与信标的相对方位。
  • 高级导航： 集成 SentenceNav 和 BrowserNav 引擎，提供流畅的文本和网页导航体验，且不与常规方向键冲突。

安装方法

  1. 从开发者的官方渠道下载最新版本的插件。
  2. 打开下载的 .nvda-addon 插件文件。
  3. NVDA 会提示您确认安装，请选择“是”(Yes)。
  4. 重启 NVDA 以完成安装。

使用说明

启用/禁用音效主题

您可以在 NVDA 的设置中启用或禁用音效主题功能：

  1. 打开 NVDA 菜单 (NVDA+N)。
  2. 进入“首选项” -> “设置”。
  3. 在设置对话框中，选择“高级音效主题”(Advanced Audio Themes) 类别。
  4. 勾选或取消勾选“启用音效主题”(Enable audio themes) 复选框。

选择与管理主题

在“音效主题”设置面板中，您可以：

  • 选择主题： 在已安装的音效主题列表中进行切换。
  • 添加新主题： 点击“添加新主题...”(Add New...) 按钮，从 .atp 文件安装主题包。
  • 删除主题： 选择一个主题并点击“删除”(Remove) 按钮。
  • 关于主题： 点击“关于”(About) 按钮以查看当前选中主题的详细信息。

使用音效主题工作室 V2

音效主题工作室允许您轻松创建和编辑主题。打开工作室的方法：

  1. 打开 NVDA 菜单 (NVDA+N)。
  2. 选择“音效主题工作室”(Audio Themes Studio)。

在工作室中，您可以：

  • 创建新音效主题： 通过向导指引您从零开始创建新主题。
  • 自定义现有主题： 修改已安装主题的各项音效。
  • 麦克风录音： 现在您可以直接通过麦克风录制声音或语音，并将其分配给 UI 事件！
  • 拖放操作 (Drag & Drop)： 将音频文件直接拖入工作室窗口即可快速完成分配。

导出您的主题

创建或编辑主题后，您可以将其导出为 .atp 文件与他人分享。导出选项位于工作室的编辑界面中。

高级语音规则与音标符号

提示音与语音规则功能允许 NVDA 播放独立的提示音 (Earcons) 以及应用其他语音效果（例如语调和语速变化）。

使用方法

  1. 确保已启用该功能。按 NVDA+Alt+P 可进行开启/关闭切换。
  2. 可以通过 NVDA 首选项菜单中的专用对话框配置各项规则。
  3. 默认情况下，插件已提供了一套预设的实用音频规则。
  4. 规则保存在 NVDA 用户配置目录下的 earconsAndSpeechRules.json 文件中。

快捷键一览

  • NVDA+Alt+N: 开启/关闭音效主题。（快速双击可开启/关闭打字音效）。
  • NVDA+Alt+T: 循环切换可用的音效主题。
  • NVDA+Alt+Y: 循环切换可用的打字音效包。
  • NVDA+Alt+K: 开启/关闭打字音效。
  • NVDA+Shift+B: 在当前导航对象位置放置/移除音频信标 (Beacon)，移动浏览时可通过声纳脉冲感知相对方位。
  • NVDA+Shift+A: 进入音效主题命令层（按下此组合键后，再按命令键：'h' 获取帮助，'t' 切换主题，'p' 切换规则，'s' 切换状态提示详细度，'c' 播报标题级别，'o' 切换语音播报顺序）。
  • NVDA+Alt+P: 开启/关闭提示音与语音规则功能。
  • NVDA+Alt+[ (左中括号): 切换简洁/详细的状态播报模式。
  • NVDA+H: 播报当前标题级别。
  • NVDA+Tab: 播报光标下对象及其完整的 3D 空间音频坐标。
  • Alt+方向键: 高级句子/短语导航 (SentenceNav)。
  • NVDA+Alt+方向键: 高级网页浏览导航 (BrowserNav)。
